## 世界观建构

### 修真界结构
- [ ] **通玄界设定**：三界之一（人间界、通玄界、仙界）
- [ ] **宗门势力**：
  - 明心剑宗（东方第一宗）：连霞山，四法三诀，连霞七剑
  - 水镜宗：琅琊水镜之天，拥有彻天水镜预知未来
  - 幽魂噬影宗：九真之首，镇宗六法门，《幽冥录》
  - 妙化宗：古音及其叔父玉散人的势力
- [ ] **通玄界格局**："十山七海三洞天，九真四异六绝地"
- [ ] **散修势力**：三大散人（血散人、玉散人等）

### 修炼体系
- [ ] **明心剑宗体系**：
  - 基础：内息搬运术 → 化气篇 → 化神篇
  - 境界：东海沉碧水 → 海上生明月 → 金丹 → 元神
  - 四法三诀：灵犀诀为最高深法门
  - 连霞七剑：天心剑、玄冥剑、洞玄剑、落霞剑、闪灵剑等
- [ ] **幽冥邪道体系**：
  - 幽明气修炼法
  - 寄魂转生之法
  - 血影妖身等邪功
- [ ] **禁制系统**：
  - 七大类：山、水、风、云、晦、明、空
  - 复合禁制：云纹+明纹=霞纹等

### 关键地点
- [ ] **连霞山**：明心剑宗总部
  - 坐忘峰：通天柱之一，高54万里
  - 启元堂：弟子修炼场所
  - 万仙台：开山弟子修行地
  - 止观峰：宗门重地，未明观
- [ ] **北极夜摩之天**：玉散人和古音的根据地
- [ ] **雾隐洞天**：李珣后期的重要活动场所

## 核心人物关系网

### 主要角色
- [ ] **李珣**：主角，福王世子，明心剑宗弟子，修炼灵犀诀
  - 性格：内敛深沉，心机深重，意志坚韧
  - 经历：八岁被血散人种下血魇，坐忘峰七年苦修
  - 身份转变：王子 → 道童 → 嫡系弟子 → 真人
- [ ] **青吟仙师**：明心剑宗前辈，与钟隐有复杂关系
  - 千年前被玉散人掳去，钟隐单剑救之
  - 在坐忘峰上隐修，教导过李珣
  - 最终飞升失败的重要推手
- [ ] **林阁**：天心剑，李珣的师父
  - 连霞七剑之首，但因古音事件意志消沉
  - 曾与天妖凤凰有感情纠葛
  - 性格颓唐，但实力深不可测

### 反派势力
- [ ] **血散人**：三大散人之一，种血魇威胁李珣
  - 要求李珣十年内获取灵犀诀
  - 修炼血魔化心大法
  - 通玄界杀孽最重的魔王
- [ ] **古音**：妙化宗宗主，七杀琴
  - 玉散人侄女，音律宗师
  - 与林阁有复杂恩怨情仇
  - 修炼太阳真火，实力强横
- [ ] **玉散人古志玄**：古魔头，隐居北极夜摩之天
  - 曾掳掠青吟，与钟隐为宿敌
  - 散修中的顶尖高手
  - 拥有众多幽玄傀儡

### 同门师友
- [ ] **单智**：李珣旧日书僮，后成嫡系弟子
  - 被明松道人收为弟子
  - 与李珣关系复杂，既有恩情又有距离
- [ ] **灵机**：李珣在低辈弟子时的好友
  - 性格纯朴善良，对李珣真心相待
  - 后去万仙台开山修炼
- [ ] **明彦道士**：明心剑宗老道，消息灵通
  - 喜欢传播江湖奇闻轶事
  - 李珣获得很多世界观信息的来源

### 情感线人物
- [ ] **水蝶兰（百幻蝶）**：李珣的道侣
  - 幻术宗师，能力神秘莫测
  - 与李珣有同心结相连
  - 关键时刻多次救助李珣
- [ ] **婴宁**：李珣收的弟子，后来背叛
- [ ] **秦婉如**：李珣在人间界的恋人，牵涉复杂阴谋
