变量更新规则:
  世界.当前时间:
    type: string
    range: 格式为 yyyy年mm月dd日 星期X 上午/下午 hh:mm(24小时制),根据实际经历的时间自然推进
    check:
      - 每次对话或场景转换后,根据实际经历的时间推进
      - 重大事件可能导致时间跨度较大
      - 保持时间的合理性和连贯性

  世界.当前地点:
    type: string
    range: 具体的房间或场所名称
    check:
      - 当角色明确移动到新地点时立即更新
      - 地点描述应具体而非笼统(如"破庙内堂"而非"某处")

  凌若雪.好感度:
    type: number
    range: capped in 0~100,初始值为0
    check:
      - 每次对话后根据凌若雪对{{user}}行为的感受更新,单次变化±1~5
      - {{user}}帮助她、保护她、尊重她会增加好感度
      - {{user}}拒绝帮助、冷漠对待、轻视她会减少好感度
      - 凌若雪性格骄傲但善良,真诚的关怀比甜言蜜语更有效

  凌若雪.当前着装:
    type: string
    range: 详细描述凌若雪当前的穿着
    check:
      - 在起床后、洗澡后、外出前、战斗后等场景需要更新
      - 反映当前的情境和活动(如战斗后衣服破损、淋雨后湿透等)
      - 保持与角色身份和性格相符

  凌若雪.当前姿势:
    type: string
    range: 描述凌若雪此刻的身体姿态和动作
    check:
      - 每次对话或场景变化时更新
      - 反映当前状态和情绪(如紧张时咬嘴唇、开心时眼睛发亮)
      - 体现角色的性格特征和当前心理状态

  凌若雪.当前想法:
    type: string
    range: 描述凌若雪当前内心的真实想法
    check:
      - 每次对话后更新,展现内心活动
      - 可能与外在表现不一致(傲娇性格)
      - 反映她对{{user}}、当前处境的真实感受

  凌若雪.关系状态:
    type: string
    range: 描述凌若雪与{{user}}的关系阶段
    check:
      - 0-20好感度："陌生人"或"初识"
      - 21-40好感度："认识的人"
      - 41-60好感度："朋友"
      - 61-80好感度："好友"或"信赖的人"
      - 81-100好感度："亲密"或"心上人"

  凌若雪.是否相遇:
    type: boolean
    range: true/false
    check:
      - 当{{user}}第一次遇到凌若雪时设置为true
      - 初始值为false,表示尚未相遇

  慕容芷.好感度:
    type: number

  慕容芷.当前姿势:

    type: array
    range: 数组,用来记录{{user}}在江湖中流传开来的事迹
    check:
      - 当{{user}}的行为被江湖中人传颂、产生影响时添加新条目

      - 例如：【雨夜救美】：某个雨夜在破庙中救下被歹徒追杀的神秘女子

  任务列表:
    type: object
    range: 对象,内部存储任务和事件,格式为{任务名,任务类型,任务描述}
    check:
      - 接受新任务或承诺参与某事件时添加新的对象
      - 不仅限于明确的"任务",包括护送、调查、寻找等待完成的事项
      - 例如：护送凌若雪回铸剑山庄、调查红尘卷书线索、寻找失踪的商队等
      - 完成任务/事件时可删除该条目
      - 任务类型分为"主线"和"支线"
      - 主线任务"寻找回家之路"始终存在,直到找到红尘卷书
